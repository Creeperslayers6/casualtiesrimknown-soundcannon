# Casualties: Rimknown - Sound Cannon
Rimworld mod that recreates the Sound Cannon from "Casualties: Unknown" inside Rimworld.
## Features:
### Turrets
- Sound Cannon (Basic) - 'Lore-accurate' version of the Sound Cannon which fires only once before needing to be serviced.
- Sound Cannon (Advanced) - Upgraded version of the Sound Cannon which can fire multiple times, but requires being powered to operate.
### Hediffs
- Internal Bleeding - Injury Hediff inflicted by shockwaves from the Sound Cannon. Results in significant bleeding in addition to severe pain.
- Hemothorax - Triggered by Internal Bleeding in the Torso. Reduces breathing capacity as severity increases.
### Surgeries
- Chest Drain - Used to remove/reset the hemothorax Hediff. Requires 1 Medicine and 3 Medical Skill to perform.
## Research Projects
- Sonic transducer : BlowbackOperation (Unlocks Basic Sound Cannon)
- Advanced sonic transducer : MicroelectronicsBasics (Unlocks Advanced Sound Cannon)
